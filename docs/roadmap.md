# Roadmap

Early-stage checklist; revise as the project gains contributors and users.

1. Stabilize `standards/cot-schema.json` with versioned releases.
2. Ship reference validators and at least one converter in `datasets/converters/`.
3. Publish a small public benchmark slice under `benchmarks/tasks/` with scoring scripts.
4. Document annotation guidelines and link them from `standards/reasoning-patterns.md`.
